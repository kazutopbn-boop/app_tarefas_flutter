# App de Tarefas

Aplicativo desenvolvido em Flutter para gerenciamento de tarefas.

## Funcionalidades

- Cadastrar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar tarefa como realizada
- Filtrar tarefas

## Tecnologias

- Flutter
- SQLite
- Provider
